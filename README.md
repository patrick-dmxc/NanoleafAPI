With this API you have the possibility to control all Nanoleaf products over the network.
In detail:
- Set
  - Current Effect
  - Effects 
  - Orientation 
  - Color
  - Brightness
- Get
  - Current Effect
  - Effects 
  - Orientation 
  - Color
  - Brightness
  - Layout
  - Firmware
  - Hardware
- Receiving of Events
  - Gestures
  - Touch
  - States
  - Layout
- Settings
  - Gestures
  - Fadings
  - Button-Control
  - Auto Brightness
- Streaming in real time Color
