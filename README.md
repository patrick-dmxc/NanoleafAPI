<h3>
With this API you have the possibility to control all Nanoleaf products over the network.
</h3>

## Supported Products:
- Light-Panels
- Canvas
- Shapes
- Essentials
- Elemants
- Lines
## Supported Features:
### Set
- Current Effect
- Effects 
- Orientation 
- Color
- Brightness
### Get
- Current Effect
- Effects 
- Orientation 
- Color
- Brightness
- Layout
- Firmware
- Hardware
### Receiving of Events
- Gestures
- Touch
- States
- Layout
### Settings
- Gestures
- Fadings
- Button-Control
- Auto Brightness
### Streaming Colors in real time
